# Objective

The objective of this repository is to have up-to-date research containing
in-depth practical and useful information about netcode and client-side
prediction techniques in multiplayer games.

When asked to add content about a specific game, framework or tool, place it
under the correct category on README.md - including comparison tables, and add a
specialized page with in-depth details at the root level. Use sub-agents for the
research process.
